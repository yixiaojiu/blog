## 根据参考图风格绘制插画 SOP

> 使用画师的作品用于 AI 生成是不道德的行为，别学我 🙏

1. 拿到一张参考图，先找到这张参考图的画师是谁（比 pixiv 的昵称为准），将画师添加到提示词中，例如：`参考图是「xxx」的一张插画`。不写这段 prompt 也没关系，只作为一个补充 Context。

   - 如果你要画立绘（图片中没有背景），选择的参考图最好只包含角色，不包含背景，图片的比例最好也和出图的比例一致。额外的背景会给模型不必要的 Context，可能会影响出图效果。

2. 让 Gemini 或者豆包描述一下参考图的艺术风格，不能是笼统的描述，例如 `2D日系动漫插画风格`。如果模型没有给出精确的描述，可以追问 `参考图的艺术风格有没有专有名词，”2D日系动漫插画风格“这种描述太宽泛了`。

3. 得到比较精确的艺术风格描述后就可以套模板生图了，以下是我常用的模板：

```
第一张图是「xxx」的一张xxx风格插画，第二张图是角色人设图。请参考图第一张图的xxx风格，为第二张图的少女画一张xxx风格插画。
【xxx风格说明】xxxx
【插画场景】xxxxxxx
【少女特征】xxxxx
【要求 (Optional)】xxxx
```

### 提示词示例

以下是根据参考图，使用文字描述绘制插画的示例。

```
参考图是「墨小斓」的一张日系萌绘（Moe-e）插画，请根据参考图的风格绘制一位少女的立绘。
【少女人设】年龄17，身高164cm，体重42kg。
【少女外貌】纯白的发色，肌肤白皙雪嫩，发梢为浅浅的淡粉色，头上有两根小小的呆毛;头发一直垂落到肩膀下面，前发处绑了一个小发带做装饰，淡蓝色渐变的瞳孔和眼睛，头发两边各用一个蓝色发带将其隔开系了一个短马尾。
【少女服设】黑白相间的魔法学院制服，中间有一个黑色的蝴蝶结，脖子处挂着一个古朴的吊坠，袖套上锈着几朵可爱的茉莉花，腰间系着一个蓝色的丝带，穿着一件白色蕾丝边的裙子，也有花朵的点缀；洁白如玉的大腿上有着可爱的白丝袜，上面绑着两个可爱的蝴蝶结，腿上穿着一双轻快的鞋子。
【日系萌绘（Moe-e）画风说明】角色拥有幼态的面部比例（大眼、小鼻、圆脸）、偏高的发际线、柔软的身体线条。通过干净、精确的赛璐璐阴影和平涂上色来定义，并配以细腻的彩色线稿。光影是高明度的，采用粉彩马卡龙色调。整体具有纯粹的治愈系（Iyashikei）氛围。
【要求】背景为白色，比例9:16，分辨率4K。
```

## 如何提升人物一致性

> Nano Banana 的人物一致性不如 Seedance2，所以会角色细节画错的情况。

1. Nano Banana 在绘制我的 OC 时，经常把帽子和发夹画错，出现这种问题的原因是模型把角色理解错了，解决办法就是对容易出错的地方补充文字说明。这个补充说明可以让 Nano Banana 生成，以下是我使用的补充说明：

```
【少女特征描述】 少女需要严格遵循人设图。少女留着米色双马尾，(两边马尾分别系着浅蓝色缎带:1.2)。她**(头上戴着一顶纯白色贝雷帽，帽子右侧带有一个白色蝴蝶结:1.2)，(左侧刘海处别着一个浅蓝色方形发夹，上面写着“TS”字样:1.3)**
```

2. 如果你抽到一张比较满意的图，但是角色的某个细节错误了，可以对这张图进行局部重绘。对错误的地方截图一张正确的参考图，再配合文字描述，让 Nano Banana 重绘出来。

## Nano Banana 平台

> 不要贪图便宜用一些第三方小网站的 API，可能会被坑（我被坑 5 美刀😭）。第三方小网站的模型可能是假的，对输入图片和输出图片的大小可能有限制(限制地很小，根本不可用)。

我只用过以下平台

- Gemini：如果你有 Gemini 的会员，可以用 Gemini 的 Pro 模型微调提示词，偶尔可以出一张好图。体感没有原生 Nano Banana 好用。
- [lovart](https://www.lovart.ai)：可以用邮箱注册，每个账号每天80积分，生成2k的图片只需10积分。个人使用下来效果不错。
- Google AI Studio：按量计费，贵，而且配置有点麻烦，一般作为我的最后选择。
