# 北路之旅

## 行程

- 10.2 从大阪到金泽，下午去湯涌温泉
- 10.3 前往 西岸站, 下午去金泽大学
- 10.4 前往 白川乡，晚上到达高山
- 10.5 在高山
- 10.6 前往砺波市，白天可以在富山市转转
- 10.7 高岗市
